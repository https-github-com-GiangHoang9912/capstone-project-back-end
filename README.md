### Create controller

```
nest g controller <Controller name>

```

### Create Service

```
nest g service <Service name>

```

### Create module

```
nest g module <Module name>

```

```
src
---cats
-------dto
----------create-cat.dto.ts
-------interfaces
----------cat.interface.ts
-------cats.controller.ts
-------cats.module.ts
-------cats.service.ts
---app.module.ts
---main.ts
```

### copy .env.example to .env
```
cp .env.example .env
```

### run server
```
npm start:dev

yarn start:dev
```
